## 使い方

### 準備するもの
1. 方言に品詞情報をアノテーションしたコーパス(翻訳用)
2. 評価するための入力文と正解文(評価用)


#### 1. 方言に品詞情報をアノテーションしたコーパス

mecab-unidic 2.1.2を参考にCSVを作る。


#### 2. 評価するための入力文と正解文(評価用)
分割性能評価(F-score)と、翻訳性能評価(Fscore,BLEU-score)を行うことができる。
